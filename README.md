# 🌸 Cherry Blossom Meadow 🌸

*A love letter disguised as a video game.*

</div>

---

A handcrafted 3D voxel world built as a personal gift — a meadow filled with cherry blossom trees, ponds, a cottage, and 12 hidden love letters waiting to be discovered. Some rest in the grass. Others sit atop sky-high parkour courses, waiting for someone brave enough to climb.

It's not a commercial project. It's a love letter in the shape of a game.

## 🎮 How to Play

| Control | Action |
|---|---|
| `WASD` | Move |
| `Mouse` | Look around (click to capture) |
| `Space` | Jump |
| `Space` × 2 | Double jump |
| `Shift` | Sprint |

Explore the meadow. Find the letters. Climb the platforms. Read what's written.

## ✨ What's in the Meadow

- **Cherry blossom trees** in four sizes, with procedurally branching trunks and pink/white/magenta blossoms
- **Four ponds** with sandy shorelines, lily pads, and water that slows you down when you wade in
- **A cottage** with a peaked roof, chimney, and a small fenced yard
- **Rocks, flowers, and grass tufts** scattered across gently rolling hills
- **Clouds** drifting lazily in a custom gradient sky
- **Butterflies** fluttering through the air
- **Fireflies** glowing softly near the ground
- **Cherry blossom petals** drifting down around you as you walk
- **12 love letters** — 7 hidden in the meadow, 5 at the tops of parkour courses
- **A full ambient soundtrack** — pad chords, a pentatonic melody, birdsong, wind, footsteps, and water sounds, all synthesized live

## 🏃 Platforming

Five letters are placed high in the sky, each one at the top of a spiral obstacle course. Platforms get smaller as you climb. Rest platforms every few steps give you a moment to breathe. A glowing heart arch marks the peak.

Double jump helps. Don't give up.

## 📖 The Journal

Your collected letters are saved in the journal (top-left corner). It tracks your progress between meadow and sky letters, and shows something special when you've found them all.

## 🛠️ Running Locally

```bash
npm install
npm run dev
```

Open [localhost:5173](http://localhost:5173).

```bash
npm run build    # Production build
npm run preview  # Preview the build
```

## 🏗️ Built With

- [Three.js](https://threejs.org/) — 3D rendering (Lambert materials, instanced terrain, PCF shadows)
- [Tone.js](https://tonejs.github.io/) — Fully synthesized audio, no audio files
- [React](https://react.dev/) — UI and component lifecycle
- [Vite](https://vitejs.dev/) — Build tooling
